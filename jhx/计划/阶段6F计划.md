# 阶段 6F 计划：渲染追踪与贴合升级（去"贴片感"）

> 2026-06-11 制定。针对两项实测反馈：①妆容跟随头部移动有明显延迟；②观感仍像"亮片贴在脸上"。只规划，不施工。

## 0. 诊断

**延迟**：渲染循环 rAF 每帧重画，但消费的关键点来自 220ms 的分析循环 + 180ms 检测节流——妆容滞后最多 ~220ms。降低帧率无效（视频实时、关键点照旧陈旧），正确方向是提高**检测**频率而非降低**渲染**频率。

**贴片感**三个来源（按破坏力排序）：
1. **遮挡不感知**：手挡嘴、刘海压眉、戴眼镜时，妆画在遮挡物上——纯 2D 多边形不知道"这块像素不是脸"；
2. **边缘是几何多边形**：羽化只能软化锯齿，软化不了"不解剖"；
3. **无姿态/光照响应**：侧脸时两侧浓度相同、唇部高光不随头部转动移动。

当前确实未使用 face parsing 与任何 3D 约束。本计划补齐前两者，3D 用轻量端侧方案（再次确认不做后端 3DDFA 重建——延迟物理学没变）。

## 1. 6F-1：每帧关键点追踪（解决延迟，优先级最高）

- 把 `detectForVideo` 从 220ms 分析循环移入 rAF 渲染循环：每帧检测 → 直接喂渲染；颜色采样、特征摘要、指标 UI 等重活留在 220ms 循环（消费最近一次每帧检测结果即可）。
- 优先尝试 MediaPipe **GPU delegate**（当前是 CPU）：初始化时探测，成功则检测耗时降一个量级；失败回落 CPU。
- 自适应降档：单帧检测耗时 > 25ms 时自动隔帧检测 + 线性插值补帧，保证低端机不雪崩。
- 加 **One-Euro 滤波**平滑关键点（每帧检测会带来轻微抖动，滤波消抖且几乎不增延迟）。
- 验收：正常速度转头/点头，妆容跟随无肉眼可见拖影（录屏逐帧检查 ≤2 帧滞后）；CPU 占用增幅可接受（任务管理器粗查 <15 个百分点）。

## 2. 6F-2：face parsing 遮挡掩膜（去贴片感主力）

复用你现有的 `人脸mask部分结构分割` 量化 ONNX：

- 接入 **onnxruntime-web**（wasm，单线程；vendor 化沿用 vosk/mediapipe 套路：下载脚本 + gitignore + /vendor/ 缓存）。
- 推理不进每帧：**低频运行**（每 400–800ms 一次，脸部 ROI 裁剪后 ~256/512 输入），输出唇/肤/眉/发等类别掩膜。
- 合成规则：最终 mask = 关键点多边形 mask **∩** parsing 类别 mask（按品类对应：口红∩唇类、眉∩眉类、眼影/腮红∩皮肤类）。两帧 parsing 之间用人脸框位移对掩膜做平移补偿。
- 效果:手挡嘴时口红从手上消失、刘海盖住的眉毛不被染色、唇角边缘贴合解剖轮廓。
- 上传照片模式跑全分辨率单次推理（顺手把 6C 的核心价值收了）。
- 降级链：onnxruntime 加载失败/推理超时 → 自动回到纯关键点 mask（现状），UI 状态行标注。
- 验收：手掌缓慢遮嘴，口红不出现在手上（容忍 parsing 频率内 <0.8s 的短暂残留）；刘海压眉场景眉色不上头发；CPU 增幅可接受。

## 3. 6F-3：轻量 3D 约束（姿态响应，锦上添花）

不做后端重建，用 MediaPipe 自带的 3D 能力：

- 打开 `outputFacialTransformationMatrixes`（当前关闭），得到头部姿态矩阵；
- 姿态响应浓度：按 yaw 对远脸侧的腮红/眼影做余弦衰减（侧脸时远侧自然变淡），唇部高光中心随头部朝向偏移；
- 利用关键点自带的 z 值微调眼影渐变方向（眼窝纵深感）；
- 明确不做：眼影立体分区渲染、睫毛/眼线、三维重光照——投入产出比留给真正需要时再评估。
- 验收：左右转头 ±30° 内，妆容浓度变化自然、无"纸片旋转"感。

## 4. 实施顺序与预算

```text
6F-1 每帧追踪 + GPU delegate + 滤波   —— 1 次会话（含降档逻辑）
6F-2 face parsing 遮挡掩膜            —— 1–2 次会话（vendor 化 + 合成 + 降级链）
6F-3 姿态响应                         —— 1 次会话（可与 6F-2 并行）
```

风险：GPU delegate 在部分核显驱动上不稳定（探测+回落兜底）；onnxruntime-web 与 MediaPipe wasm 并存的内存占用需实测（必要时 parsing 降到 256 输入）；One-Euro 参数需真人调优一次。

## 5. 验收总场景

开试妆四品类齐上 → 正常语速说话 + 转头：妆容贴脸不拖影、嘴部张合牙齿干净、手挡嘴妆不上手、侧脸浓度自然——四条全过，"亮片感"才算真正解决。
